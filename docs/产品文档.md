# 喵 · 产品文档

> 版本：v2.0 · 日期：2026-07-12

---

## 一、产品概述

**喵（Miao）** 是一款以猫娘为主题、以温馨陪伴为核心体验的 AI 聊天 Android 应用，配套独立后台管理系统。

用户通过账号密码登录，无需配置任何 API Key，即可与多家主流 AI 大模型对话。所有聊天记录、记忆片段、个人设置均存储在云端，多设备无缝同步。后台管理端由运营人员统一管理 API Key、用户账户和系统配置。

---

## 二、系统组成

```
喵 系统
├── Android App（用户端）
│   └── 账号登录 → 云端数据 → AI 对话 → 记忆管理
├── 后台服务器（API Server）
│   └── 用户认证 / AI 请求代理 / 数据存储 / 记忆检索
└── 后台管理端（Web Admin）
    └── 用户管理 / API Key 管理 / 模型配置 / 数据监控
```

## 三、目标用户

| 用户群体 | 特征描述 |
|---|---|
| 二次元爱好者 | 喜欢猫娘、萌文化，寻求情感陪伴 |
| 轻度 AI 用户 | 不想折腾 API Key，开箱即用 |
| 重度聊天用户 | 需要跨设备同步、长期记忆、历史查阅 |
| 运营 / 管理员 | 通过后台管理全体用户和模型资源 |

---

## 四、产品定位

- **零配置体验**：用户注册即用，API Key 由运营方在后台统一管理，用户不感知。
- **云端长期记忆**：AI 能记住用户的喜好、习惯、重要事件，跨会话持续积累。
- **情感陪伴优先**：猫娘人格贯穿整个交互，有温度的伙伴而非冷冰冰的工具。
- **多模型无缝切换**：同一账号下，Claude / DeepSeek / Qwen / GLM / OpenAI 随意切换。

---

## 五、核心功能模块

| 模块 | 描述 |
|---|---|
| 账号系统 | 注册 / 登录 / 找回密码，JWT 鉴权，多设备同步 |
| 多会话管理 | 微信风格聊天列表，会话数据全量云同步 |
| AI 对话 | 服务器代理 AI 请求，流式 SSE 透传，支持 5 家模型 |
| 云端记忆系统 | 用户可手动添加记忆片段，AI 自动提取关键记忆，语义检索注入上下文 |
| 数据云同步 | 聊天记录、设置、人格配置均存储在服务器 |
| 后台管理端 | Web 端，管理 API Key、用户账户、模型开关、系统监控 |

## 六、商业模式

| 版本 | 用户费用 | 说明 |
|---|---|---|
| 免费版 | 免费 | 每日消息配额限制（如 50 条/天） |
| 会员版（规划） | 订阅制 | 无限消息、优先队列、更多模型选项 |
| 运营方 | 承担 API 成本 | 后台统一管控 API Key 用量和成本 |

## 八、版本路线图
